# ABABIL

ABABIL is a high-performance, EVM-compatible Layer-1 blockchain built with the Cosmos SDK.

## Features

- EVM Compatible
- Solidity Smart Contracts
- MetaMask Support
- Native Coin: ABABIL (ABL)
- Fast Block Time (Target: 1 Second)
- Low Transaction Fees
- Proof of Stake (PoS)
- Cosmos SDK + CometBFT

## Network

- Chain Name: ABABIL
- Chain ID: ababil-1
- Native Coin: ABL
- Smallest Unit: uabl

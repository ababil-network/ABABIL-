# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| Main | ✅ Yes |
| Testnet | ✅ Yes |
| Older Versions | ❌ No |

## Reporting a Vulnerability

The ABABIL team takes security seriously.

If you discover a security vulnerability, please do not create a public GitHub issue.

Instead, report it privately by email.

Email:
security@ababil.network

Please include:

- Description of the issue
- Steps to reproduce
- Possible impact
- Suggested fix (if available)

We will acknowledge your report as soon as possible and work on a fix.

Thank you for helping keep ABABIL secure.
